# 03. Aprobación Explícita

- Para cualquier cambio de naturaleza estructural, creación de nuevos flujos o modificaciones de contenido que alteren la estrategia del plan (ej: cambiar precios, alterar el target, modificar el modelo de servicio), el agente **debe detenerse y solicitar la aprobación explícita** del usuario antes de proceder.
- La confirmación no debe ser asumida bajo ninguna circunstancia.
- Para requerir aprobación, se debe indicar claramente la acción propuesta, los archivos impactados, y esperar una respuesta afirmativa como "Aprobado".
