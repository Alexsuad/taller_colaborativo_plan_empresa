# Gate transversal — Producto, marketing y comunicación

## Propósito
Bloquear el avance de marketing, ventas, comunicación y redes si no existe coherencia suficiente con el cliente y la propuesta de valor.

## Validación mínima
| Criterio | Evidencia | Estado |
| --- | --- | --- |
| Cliente objetivo definido | Segmento o perfil claro | OK / BLOQUEADO |
| Segmentación clara | Segmento distinguible | OK / BLOQUEADO |
| Propuesta de valor específica | Oferta concreta | OK / BLOQUEADO |
| Oferta al mercado clara | Qué se vende y a quién | OK / BLOQUEADO |
| Canales coherentes | Canal ligado al cliente | OK / BLOQUEADO |
| Comunicación conectada | Mensaje alineado | OK / BLOQUEADO |
| Redes con función definida | Rol de cada red | OK / BLOQUEADO |
| Calendario accionable | Cadencia realista | OK / BLOQUEADO |
| No genérico | Sin humo ni vaguedad | OK / BLOQUEADO |

## Dictamen
- `APTO_PARA_REVISION_DE_CIERRE_COMERCIAL`
- `MANTENER_BLOQUEADO`
- `REQUIERE_AJUSTES`

## Regla
Si cualquier criterio clave queda en BLOQUEADO, no avanzar a redacción ni a gates posteriores de esa línea.
