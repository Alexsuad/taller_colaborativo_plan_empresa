# Gate transversal — Producto, marketing y comunicación

## Propósito
Bloquear el avance de marketing, ventas, comunicación y redes si no existe coherencia lógica, recursos suficientes, KPIs y viabilidad operativa en toda la cadena comercial con el cliente y la propuesta de valor.

## Alcance del Gate
- Este gate valida la coherencia comercial del bloque auditado.
- No valida la viabilidad global del negocio.
- No sustituye la revisión económico-financiera.
- No autoriza apertura de la Fase 4.
- No autoriza la redacción global ni la apertura global del Gate 1D.

## Criterios de Validación Mínima Obligatoria

| Criterio | Evidencia requerida | Estado |
| --- | --- | --- |
| **Cliente objetivo definido** | Segmento o perfil de cliente acotado (no generalista). | OK / BLOQUEADO |
| **Diferenciación de roles** | Distinción explícita entre cliente pagador y usuario del taller. | OK / BLOQUEADO |
| **Problema o necesidad** | Identificación de dolores reales y la alternativa actual del cliente. | OK / BLOQUEADO |
| **Propuesta de valor específica** | Oferta diferencial y justificación del resultado de valor obtenido. | OK / BLOQUEADO |
| **Canales priorizados** | Clasificación de canales en propios, ganados y pagados coherentes con el cliente. | OK / BLOQUEADO |
| **Acciones y presupuesto** | Plan de acciones comerciales con presupuesto de marketing asignado y realista. | OK / BLOQUEADO |
| **Capacidad de ejecución** | El plan no supera el tiempo y capacidad física de los recursos del taller. | OK / BLOQUEADO |
| **Proceso comercial definido** | Flujo del embudo de ventas, objeciones típicas y argumentos detallados. | OK / BLOQUEADO |
| **Activos comerciales** | Herramientas de venta mínimas descritas (catálogo, web, cobros). | OK / BLOQUEADO |
| **Política de cobro** | Plazos de cobro explícitos y alineados con el modelo de ingresos. | OK / BLOQUEADO |
| **Comunicación y tono** | Mensajes sencillos y tono consistente apoyados por pruebas de confianza. | OK / BLOQUEADO |
| **Redes con función subordinada** | Cada red social tiene asignado un rol específico que apoya a la estrategia. | OK / BLOQUEADO |
| **Existencia de KPIs** | Indicadores clave de rendimiento (KPIs) definidos para cada fase del embudo. | OK / BLOQUEADO |
| **No genérico (sin humo)** | Sin vaguedad conceptual ni promesas de crecimiento sin base de datos. | OK / BLOQUEADO |

## Reglas de Bloqueo Explícitas

- **Bloqueo por sustitución táctica:** Queda estrictamente **BLOQUEADO** el avance si el plan confía toda la atracción y conversión comercial en redes sociales, sustituyendo la estrategia de ventas directa o los canales de captación offline prioritarios.
- **Bloqueo por falta de conexión comercial:** Queda **BLOQUEADO** si se planifican campañas de marketing sin detallar el proceso comercial de conversión de esos leads a clientes de pago.
- **Bloqueo por sobrecarga de recursos:** Se vetará el avance si el volumen de acciones de marketing o publicaciones de redes excede las horas estimadas de dedicación del personal del taller sin justificación de presupuesto externo.
- **Bloqueo por falta de KPIs:** El gate se mantendrá bloqueado si alguna sección no cuenta con sus correspondientes métricas medibles de éxito.

## Dictámenes Válidos (Fase 2)
El gate solo podrá resolver con uno de los siguientes estados. Queda estrictamente prohibido aprobar transiciones o aperturas de la Fase 4:

- `APTO_PARA_REVISION_DE_CIERRE_COMERCIAL`
- `REQUIERE_AJUSTES`
- `MANTENER_BLOQUEADO`
