## 1. Información útil para Fase 1

### 1.1 Modelo observado

The CoMAKING Space funciona como un referente de **coworking industrial-artesanal con pago por uso**.

Su modelo se diferencia de Cara y Canto y de A tu Madera porque no parece basarse principalmente en cursos ni en mensualidades altas, sino en una combinación muy interesante:

```text
membresía baja + alta inicial + pago por horas + bonos + almacenamiento opcional
```

Para Zaragoza, este modelo es especialmente útil porque reduce la barrera psicológica de entrada. Un usuario no necesita comprometerse desde el principio con una cuota mensual alta. Puede pagar una membresía baja, usar el taller según necesidad y escalar si el uso aumenta.

### 1.4 Modelo de acceso

El modelo observado se puede resumir así:

```text
alta inicial + membresía baja + pago por uso
```

Esto permite separar dos cosas:

* pertenecer al ecosistema;
* pagar realmente por el uso del taller.

Para Zaragoza, esta lógica puede ser muy valiosa porque evita dos errores:

1. cobrar mensualidades altas antes de que exista confianza;
2. regalar demasiadas horas dentro de una cuota baja.

El usuario paga una entrada accesible, pero el coste variable aparece cuando usa el taller.

---

### 1.5 Bonos y pago por uso

The CoMAKING Space valida una escalera de uso muy clara:

* usuario puntual que reserva pocas horas;
* usuario recurrente que accede a mejores precios por tramos;
* miembro con bono;
* usuario intensivo con bono grande;
* usuario que necesita almacenamiento.

La idea más útil para Zaragoza no es copiar cada tarifa, sino copiar la lógica:

```text
cuanto más uso validado, mejor precio por hora
```

Esto permite captar usuarios ocasionales sin perder la posibilidad de fidelizar a usuarios recurrentes.

---

### 1.6 Cursos y capacitación

La capacitación aparece como elemento importante del modelo, aunque la fuente revisada no permite reconstruir todo el sistema pedagógico.

La lección aplicable para Zaragoza es clara:

```text
el pago por uso solo funciona si el acceso está ordenado por niveles de competencia
```

No basta con cobrar por horas. Hay que saber quién puede usar qué, en qué condiciones y con qué grado de supervisión.

---

### 1.7 Seguridad y prevención

The CoMAKING Space es especialmente relevante porque las fuentes internas indican que publica elementos como evaluación de riesgos, plan de emergencia y legalización de actividad.

Esto es muy importante para Zaragoza.

El aprendizaje principal es:

```text
la prevención no es decoración documental;
es parte central del modelo operativo
```

Un taller compartido con usuarios externos necesita:

* inducción;
* normas de uso;
* control de maquinaria;
* reserva;
* supervisión;
* EPIs;
* mantenimiento;
* limpieza;
* extracción de polvo;
* trazabilidad de usuarios;
* protocolos de emergencia;
* seguro compatible;
* legalización de actividad.

---

### 1.8 Maquinaria

Las fuentes internas mencionan:

* maquinaria industrial HAMMER;
* herramientas FESTOOL;
* maquinaria de carpintería;
* conexión con herrería / soldadura.

Para Zaragoza, la maquinaria industrial de carpintería es una referencia útil, pero debe filtrarse.

La Fase 1 no debe copiar una infraestructura demasiado amplia. El proyecto Zaragoza debe centrarse en:

* bancos de trabajo;
* herramienta manual;
* herramientas electroportátiles básicas;
* maquinaria estacionaria convencional imprescindible;
* aspiración/extracción;
* EPIs;
* mantenimiento y seguridad.

---

### 1.9 Comunidad y eventos

CoMAKING también aparece asociado a comunidad maker, eventos, actividades y team building.

Esto aporta una línea interesante para fase futura o para ingresos complementarios, pero no debería distraer el arranque.

Para Zaragoza, puede usarse como referencia secundaria para:

* talleres experienciales;
* eventos de empresa;
* actividades grupales;
* workshops temáticos;
* jornadas de puertas abiertas;
* alianzas con colectivos creativos.

---

### 1.10 Almacenamiento

La estantería privada opcional de 15 €/mes es una referencia muy útil.

A diferencia de A tu Madera, donde el almacenamiento aparece integrado en planes más amplios, CoMAKING lo separa como extra sencillo.

Esto puede ser muy aplicable a Zaragoza:

```text
almacenamiento como extra simple, no como obligación dentro de todos los planes
```

## 3. Información contaminante para Fase 1

### 3.1 Herrería y soldadura

* **Elemento contaminante:** conexión con herrería / soldadura.
* **Motivo:** Abre demasiado el foco operativo, legal, preventivo y de maquinaria.
* **Acción:** `DESCARTAR_FASE_1`
* **Posible recuperación futura:** Solo como línea futura si existe espacio, demanda, seguros, permisos y personal especializado.

---

### 3.2 Camperización

* **Elemento contaminante:** orientación a proyectos de camperización.
* **Motivo:** Puede requerir mucho espacio, patio exterior, logística de vehículos, seguros específicos y tiempos largos de ocupación.
* **Acción:** `DESCARTAR_FASE_1 / FASE_FUTURA_CONDICIONAL`
* **Uso adaptado:** Proyectos pequeños de madera para hogar o mobiliario, no vehículos completos.

---

### 3.3 Amplitud maker-industrial

* **Elemento contaminante:** coworking industrial demasiado amplio.
* **Motivo:** Puede llevar el proyecto hacia makerspace industrial, no hacia coworking artesanal de carpintería.
* **Acción:** `ADAPTAR_FASE_1`

Para Zaragoza debe mantenerse el foco:

```text
madera artesanal + formación + restauración + ebanistería + uso controlado de taller
```

---

### 3.4 Maquinaria demasiado profesional desde el inicio

* **Elemento contaminante:** copiar una dotación industrial de alta gama.
* **Motivo:** Puede disparar CAPEX, mantenimiento, seguros, formación y riesgo operativo.
* **Acción:** `ADAPTAR_FASE_1`

La referencia sirve para entender aspiración de calidad, no para sobredimensionar la inversión inicial.

## 4. Información útil para fase futura

### 4.1 Bonos grandes

El bono de 100h puede ser útil para una fase posterior, cuando haya usuarios intensivos o semi-profesionales.

Condición para reconsiderarlo:

* suficiente demanda recurrente;
* sistema de reservas maduro;
* control de capacidad;
* márgenes claros;
* maquinaria con mantenimiento previsto.

---

### 4.2 Eventos y team building

CoMAKING demuestra que el espacio puede monetizarse también con actividades grupales.

Posible aplicación futura:

* talleres corporativos;
* experiencias de equipo;
* jornadas de iniciación;
* actividades con colegios o entidades;
* workshops de economía circular;
* eventos de comunidad.

Condición: no distraer la operación básica ni convertir el taller en espacio de eventos antes de validar el núcleo.

---

### 4.3 Almacenamiento modular

La estantería privada como extra puede ser una línea futura o inicial si el local lo permite.

Aplicación posible:

* estantería mensual;
* cajón o taquilla;
* espacio de proyecto temporal;
* zona de madera identificada por usuario.

Condición: reglas estrictas de volumen, tiempo, seguridad, limpieza y materiales permitidos.

## 6. Síntesis de la fuente

### 6.1 Qué aporta realmente

The CoMAKING Space aporta una referencia muy potente para una Fase 1 basada en acceso flexible.

Su mayor aprendizaje es:

```text
no hace falta obligar al usuario a una cuota alta;
puede entrar con una membresía baja y pagar según uso real
```

Esto es especialmente interesante para Zaragoza, porque permite validar demanda sin exigir compromiso mensual fuerte desde el primer contacto.

También aporta una advertencia muy seria: un taller compartido no se sostiene solo con precios. Necesita prevención, legalización, mantenimiento, reservas, orden y control operativo.

---

### 6.2 Qué no aporta

No debe usarse como copia completa del modelo Zaragoza porque:

* opera en un entorno metropolitano cercano a Barcelona;
* mezcla más oficios de los que convienen en Fase 1;
* incluye lógica industrial y maker más amplia;
* puede atraer proyectos de gran tamaño como camperización;
* su dotación de maquinaria puede ser superior a lo prudente para el arranque.

---

### 6.3 Cómo debe usarse en la matriz

The CoMAKING Space debe entrar como referente de **alta aplicabilidad** para:

* membresía baja;
* pago por uso;
* bonos por horas;
* almacenamiento opcional;
* capacitación;
* prevención documentada;
* segmentación entre usuario puntual y usuario intensivo.

Debe filtrarse para no copiar directamente:

* herrería;
* soldadura;
* camperización;
* amplitud maker-industrial;
* maquinaria excesiva;
* eventos corporativos como prioridad inicial.

---

## 7. Conclusión operativa

The CoMAKING Space debe clasificarse como:

```text
REFERENTE_NUCLEO
```

para el diseño del modelo comercial flexible de Zaragoza.

No debe copiarse como coworking industrial amplio, sino como referencia de una idea concreta:

```text
entrada barata,
uso medido,
pago por horas,
bonos escalables,
almacenamiento como extra,
seguridad documentada,
crecimiento gradual.
```

Para Zaragoza, CoMAKING es especialmente útil para diseñar el componente de **pago por uso**, mientras que Cara y Canto sirve mejor para **formación + bonos controlados** y A tu Madera para una **escalera comercial más completa**.
