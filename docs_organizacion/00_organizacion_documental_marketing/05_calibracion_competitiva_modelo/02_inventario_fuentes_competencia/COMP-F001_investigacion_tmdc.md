## 1. Información útil para Fase 1

### 1.1 Modelo observado

TMDC funciona como un referente de **coworking industrial de fabricación**, no como un taller artesanal pequeño.

Su modelo se apoya en:

* acceso compartido a infraestructura industrial;
* comunidad de fabricantes;
* maquinaria para madera, metal, plásticos y fabricación digital;
* membresías;
* sistema de créditos;
* capacitación para uso de maquinaria;
* espacios profesionales;
* apoyo a proyectos de diseño, fabricación y emprendimiento.

Para Zaragoza, TMDC no debe leerse como modelo de arranque, sino como referente de madurez.

La lección principal es:

```text
Un taller compartido puede convertirse en infraestructura productiva seria,
pero eso exige escala, control, prevención, comunidad y gestión profesional.
```

TMDC aparece asociado a:

* taller industrial compartido;

---

### 1.4 Modelo de acceso

El patrón observado es:

```text
membresía + créditos + coste por recurso
```

Esto permite facturar de forma diferenciada según el tipo de uso:

* no cuesta lo mismo usar banco que maquinaria industrial;
* no cuesta lo mismo montar que cortar, soldar o mecanizar;
* los usuarios intensivos pagan más;
* la maquinaria cara se monetiza de forma proporcional a su uso.

Para Zaragoza, esta lógica es interesante a futuro, pero demasiado compleja para Fase 1.

En el arranque conviene una estructura más simple:

```text
curso / inducción + bono + pago por uso simple + almacenamiento opcional
```

---

### 1.5 Capacitación y seguridad

TMDC es útil como referencia de control de acceso a maquinaria.

Las fuentes internas indican que exige visita previa y capacitación para usar maquinaria industrial. También aparece una lógica de inducción diferenciada: inducción rápida para quien ya tiene experiencia y formación más profunda para quien no ha usado las máquinas.

Esto valida una regla clave para Zaragoza:

```text
La maquinaria no puede estar disponible solo por pagar.
Debe existir autorización previa.
```

Aplicaciones para Fase 1:

* inducción obligatoria;
* prueba mínima de competencia;
* autorización por tipo de máquina;
* registro de usuarios autorizados;
* supervisión inicial;
* normas de seguridad visibles;
* EPIs obligatorios;
* mantenimiento documentado;
* reserva de maquinaria;
* separación entre banco, herramienta manual y maquinaria estacionaria.

---

### 1.6 Comunidad

TMDC aporta una evidencia importante: la comunidad no es solo decoración de marca.

En este tipo de espacios, la comunidad puede generar:

* colaboración entre perfiles;
* intercambio de conocimientos;
* acceso a especialistas;
* proyectos compartidos;
* retención de usuarios;
* reputación;
* aprendizaje informal.

Para Zaragoza, esta idea sí es aprovechable, pero debe aterrizarse en un formato más pequeño:

* comunidad de usuarios;
* exalumnos;
* profesionales invitados;
* restauradores;
* aficionados avanzados;
* jornadas de puertas abiertas;
* mentorías puntuales;
* proyectos colaborativos controlados.

---

### 1.7 Reconocimiento e impacto

La investigación interna presenta a TMDC como un referente reconocido por su impacto, sostenibilidad y economía circular, incluyendo validación institucional europea.

Esto no debe usarse para inflar artificialmente el proyecto de Zaragoza, pero sí como inspiración:

```text
El acceso compartido a infraestructura productiva puede tener impacto económico,
social y ambiental si se gestiona con rigor.
```

Para Zaragoza, la lectura correcta es prudente:

* no copiar la escala;
* sí aprender del discurso de acceso compartido;
* sí aprender del enfoque de circularidad;
* sí aprender de la profesionalización;
* sí aprender del papel como infraestructura para emprendedores.

## 3. Información contaminante para Fase 1

### 3.1 Fabricación digital

* **Elemento contaminante:** CNC, láser, impresión 3D, cabina de pintura avanzada y recursos digitales.
* **Motivo:** Ya se decidió que la Fase 1 del proyecto Zaragoza no debe incluir fabricación digital ni maquinaria tecnológica avanzada en el análisis financiero inicial.
* **Acción:** `DESCARTAR_FASE_1`
* **Posible recuperación futura:** Solo como fase futura condicional y con demanda validada.

---

### 3.2 Metal, soldadura y plásticos

* **Elemento contaminante:** apertura a múltiples materiales y oficios industriales.
* **Motivo:** Aumenta permisos, riesgos, seguros, layout, prevención, inversión y complejidad operativa.
* **Acción:** `DESCARTAR_FASE_1`
* **Posible recuperación futura:** Fase 2 o 3, nunca como núcleo inicial.

---

### 3.3 Sistema de créditos

* **Elemento contaminante:** modelo de membresía + créditos desde el inicio.
* **Motivo:** Puede ser demasiado difícil de explicar, administrar y vender en un mercado inicial no validado.
* **Acción:** `ADAPTAR_FASE_FUTURA`

Para Zaragoza conviene empezar con precios comprensibles:

* sesión;
* bono;
* curso;
* hora de maquinaria;
* almacenamiento;
* cuota simple.

---

### 3.4 Escala industrial

* **Elemento contaminante:** replicar escala de miles de metros cuadrados, múltiples sedes o infraestructura pesada.
* **Motivo:** Dispara CAPEX, personal, mantenimiento, seguros, marketing, complejidad legal y punto de equilibrio.
* **Acción:** `DESCARTAR_FASE_1`

---

### 3.5 Acceso 24/7

* **Elemento contaminante:** acceso autónomo extendido.
* **Motivo:** En un taller con maquinaria y usuarios externos, el acceso 24/7 aumenta el riesgo si no hay madurez operativa.
* **Acción:** `FASE_FUTURA_CONDICIONAL`

Condiciones mínimas para reconsiderarlo:

* usuarios profesionales validados;
* seguro compatible;
* control de acceso;
* cámaras;
* registro de uso;
* protocolos de emergencia;
* historial de uso seguro;
* mantenimiento estable.

---

## 4. Información útil para fase futura

### 4.1 Sistema de créditos por recurso

Puede ser útil cuando el proyecto tenga:

* suficiente volumen de usuarios;
* diferentes tipos de maquinaria;
* datos reales de uso;
* software o sistema de reservas;
* capacidad administrativa;
* estructura de mantenimiento;
* segmentación clara entre usuarios básicos y profesionales.

---

### 4.2 Comunidad profesional avanzada

La comunidad de fabricantes puede ser una aspiración futura:

* diseñadores;
* restauradores;
* ebanistas;
* carpinteros;
* tapiceros;
* arquitectos;
* interioristas;
* estudiantes;
* makers filtrados;
* pequeños emprendedores.

Pero para Fase 1 debe construirse desde lo pequeño: cursos, talleres, exalumnos y usuarios recurrentes.

---

### 4.3 Programas de incubación o talento joven

TMDC muestra que el taller compartido puede conectar con:

* ayudas públicas;
* programas de jóvenes diseñadores;
* innovación social;
* economía circular;
* sostenibilidad;
* emprendimiento creativo.

Esto puede ser muy útil para convocatorias y alianzas en fases posteriores.

---

### 4.4 Expansión de materiales

Metal, soldadura, plásticos o fabricación digital pueden considerarse más adelante, pero solo si no diluyen la identidad del proyecto.

La secuencia correcta sería:

```text
madera artesanal → restauración / ebanistería / tapicería ligera →
servicios complementarios → alianzas externas →
nuevas líneas técnicas solo si hay demanda validada
```

## 6. Síntesis de la fuente

### 6.1 Qué aporta realmente

TMDC aporta una visión de largo plazo.

Muestra que un taller compartido puede convertirse en:

* infraestructura productiva seria;
* comunidad profesional;
* plataforma de emprendimiento;
* espacio de economía circular;
* red de aprendizaje;
* recurso para diseñadores y fabricantes.

Su principal valor para Zaragoza es demostrar lo que el proyecto podría llegar a ser si madura, no lo que debe ser el primer día.

---

### 6.2 Qué no aporta

TMDC no aporta un modelo simple de arranque para Zaragoza.

No debe usarse para justificar:

* inversión inicial sobredimensionada;
* compra de CNC;
* láser;
* impresión 3D;
* soldadura;
* metal;
* créditos complejos;
* acceso 24/7;
* gran nave industrial desde el inicio;
* estructura de personal pesada.

Copiar TMDC demasiado pronto sería un error financiero y operativo.

---

### 6.3 Cómo debe usarse en la matriz

TMDC debe entrar como referente de **aplicabilidad media para Fase 1** y **alta utilidad para fase futura**.

Debe usarse para aprender sobre:

* comunidad de fabricantes;
* profesionalización;
* control de maquinaria;
* sistema avanzado de monetización;
* economía circular;
* relación con innovación y emprendimiento;
* escalabilidad.

Debe filtrarse para no copiar:

* fabricación digital;
* metal y soldadura;
* complejidad de créditos;
* escala industrial;
* acceso 24/7;
* inversión pesada inicial.

---

## 7. Conclusión operativa

TMDC debe clasificarse como:

```text
REFERENTE_SECUNDARIO / REFERENTE_FUTURO
```

No como referente núcleo del MVP.

La lectura correcta para Zaragoza es:

```text
TMDC inspira la visión de madurez,
pero no define el arranque.
```

Para la Fase 1, son más útiles Cara y Canto, CoMAKING y A tu Madera. TMDC debe quedar como referencia para pensar crecimiento, profesionalización, comunidad, prevención y monetización avanzada cuando el modelo básico ya esté validado.
