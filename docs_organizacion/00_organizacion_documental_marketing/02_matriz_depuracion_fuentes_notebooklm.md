# 02_Matriz de depuración de fuentes NotebookLM

## Fase 0.0 — Organización documental de marketing

**Proyecto:** Taller Colaborativo Artesanal en Zaragoza  
**Fuente base:** NotebookLM — `Taller Colaborativo Zaragoza — Organización Documental Marketing`  
**Estado:** Matriz de depuración documental

---

## 1. Objetivo

Identificar fuentes repetidas, solapadas, contradictorias, contaminadas o pendientes de verificación antes de usarlas en documentos posteriores.

Este archivo sirve para:

- evitar duplicidades;
- diferenciar fuentes primarias y secundarias;
- detectar contradicciones;
- marcar contenido contaminado;
- priorizar revisiones documentales.

No autoriza eliminar fuentes automáticamente.  
No redacta el plan de empresa.  
No cierra decisiones de negocio.

---

## 2. Nota de interpretación

Una fuente “repetida”, “similar” o “solapada” no significa que deba eliminarse.

Significa que comparte tema, datos o enfoque con otra fuente.

Antes de descartar cualquier documento, se debe revisar qué aporta de forma única:

- datos primarios;
- análisis estratégico;
- contexto histórico;
- narrativa;
- advertencias;
- precios;
- metodología;
- interpretación.

Las fuentes primarias prevalecen para datos verificables:

- tarifas;
- servicios;
- horarios;
- condiciones;
- normativa;
- información oficial.

Las fuentes secundarias pueden conservarse para:

- análisis;
- lectura estratégica;
- riesgos;
- comparaciones;
- aplicabilidad al proyecto de Zaragoza.

## 8. Regla final

Esta matriz no decide qué se elimina.

Sirve para saber qué revisar primero y con qué criterio.

```text
Solapado no significa eliminable.
Contradictorio no significa descartable.
Histórico no significa inútil.
Secundario no significa prescindible.
```
