# 00_Plan de implementación — Fase 0.0

## Organización documental de marketing

**Proyecto:** Taller Colaborativo Artesanal en Zaragoza
**Carpeta:** `docs_organizacion/00_organizacion_documental_marketing/`
**Estado:** Documento de control interno

---

## 1. Objetivo

Organizar las fuentes de marketing antes de redactar el plan de empresa.

Esta fase sirve para:

* saber qué fuentes existen;
* resumir qué contiene cada fuente;
* identificar qué fuente sirve y para qué;
* detectar fuentes repetidas, solapadas o contradictorias;
* separar fuentes útiles, históricas, contaminadas o pendientes de verificar;
* definir qué falta antes de escribir el plan.

---

## 2. Qué no se hace en esta fase

En esta fase no se debe:

* redactar el plan de empresa;
* completar `respuestas_plan_empresa/`;
* crear documentos consolidados finales;
* cerrar ubicación;
* cerrar precios;
* cerrar forma jurídica;
* cerrar viabilidad;
* cerrar cronograma;
* usar fuentes sin revisar como contenido definitivo.

---

## 3. Regla Lean / 5S

Solo se conserva o crea documentación que ayude a ordenar, decidir o reducir dudas.

No se conserva:

* documentación duplicada sin valor;
* placeholders innecesarios;
* archivos creados antes de tiempo;
* documentos que confunden el flujo;
* material que no aporta a esta fase.

Regla práctica:

```text
Menos documentos, más claridad.
```

---

## 4. Carpeta activa

La única carpeta activa de esta fase es:

```text
docs_organizacion/00_organizacion_documental_marketing/
```

No se deben tocar otros archivos o carpetas del repositorio sin autorización explícita.

---

## 5. Archivos y carpetas activos actuales

La carpeta debe contener solo estos elementos:

```text
00_plan_implementacion_fase_0_0.md
01_inventario_descriptivo_fuentes_notebooklm.md
02_matriz_depuracion_fuentes_notebooklm.md
04_extracciones_documentales_notebooklm/
```

---

## 6. Función de cada elemento

| Elemento | Función |
| :--- | :--- |
| `00_plan_implementacion_fase_0_0.md` | Define qué se hace, qué no se hace y qué archivos existen en esta fase. |
| `01_inventario_descriptivo_fuentes_notebooklm.md` | Resume las fuentes activas de NotebookLM y su posible utilidad. |
| `02_matriz_depuracion_fuentes_notebooklm.md` | Detecta solapes, contradicciones, contaminación y prioridades de revisión. |
| `04_extracciones_documentales_notebooklm/` | Sede que guarda las extracciones amplias de NotebookLM por cada bloque del mapa 03. Sirve como material documental de trabajo para el equipo de marketing/redacción, no es redacción final del plan, no contiene análisis de marketing y no sustituye los archivos 01, 02 ni 03. |

---

## 7. Orden de trabajo

El orden correcto es:

```text
1. Crear este documento de control.
2. Crear el inventario descriptivo de fuentes.
3. Crear la matriz de depuración.
4. Revisar si hace falta crear documentos adicionales.
```

No se crean más archivos hasta completar y revisar `01` y `02`.

---

## 8. Criterio de cierre

La fase se puede cerrar cuando:

* las fuentes estén inventariadas;
* cada fuente tenga resumen breve;
* cada fuente tenga uso preliminar;
* estén detectados solapes y contradicciones;
* estén marcadas las fuentes contaminadas o históricas;
* estén identificados los vacíos principales;
* no se haya redactado el plan de empresa.

---

## 9. Frase de control

```text
No estamos escribiendo el plan.
Estamos ordenando las fuentes para poder escribirlo bien después.
```
